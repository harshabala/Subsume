# Chrome Web Store promotional assets

Place final store images here before packaging a listing upload. Extension toolbar icons live in `src/assets/icons/` and are copied into `dist/icons/` at build time (16 / 48 / 128).

## Required / common sizes

| Asset | Size (px) | Notes |
|-------|-----------|--------|
| Store icon | **128×128** | Same art as `icon128.png`; listing may also accept larger square masters |
| Small promo tile | **440×280** | Optional but recommended |
| Marquee promo tile | **1400×560** | Optional; large listing banner |
| Screenshots | **1280×800** or **640×400** | At least one; up to 5. Prefer 1280×800 |
| Promo video | YouTube URL | Optional |

## Suggested content for screenshots

1. Sanctuary / home journal view  
2. Capture / reflection flow  
3. People / filmmakers follow  
4. Discovery while browsing (dock or hover card)  
5. Settings with Drive backup connected (privacy-minded)

## Brand

See repo root `brand.md` and `CINEMATIC_JOURNAL_DESIGN_SPEC.md` for tone, color, and copy. Avoid cluttered UI chrome; prefer cinematic stills and short captions.
