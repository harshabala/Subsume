# Subsume

**What:** Chrome extension for film/TV discovery, library, capture, and sanctuary-style browsing.

**Who:** Enthusiasts curating a personal media library with optional cloud backup.

**Register:** Product UI (settings, tables, extension shell). Visual system: Gilded Night / Cinematic Sanctuary (`src/shared/tokens.css`).

**Primary tasks:** Browse library, capture moments, configure API keys, connect Google Drive backup, read diagnostics when something fails.

**Tone:** Calm, literary dark UI; gold accent for selection and primary actions only.
