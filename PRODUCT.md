# Subsume — product context (Impeccable)

**register:** product  
**surface:** Chrome extension full-page UI (Sanctuary library, Discovery, Settings). Shadow DOM / cinematic journal aesthetic.

## Users
Cinephiles who track films privately; reflection-first, not metrics-first.

## Tasks
- Browse sanctuary (watched / queue / memory intents)
- Read and expand emotional reflections (~120ch on cards)
- Configure taste, APIs, AI curator, backup
- Discover via search, recommendations, stats

## Design constraints
- Dark cinematic themes (tokens in `sanctuary.css`); no marketing landing patterns
- One navigation system (primary + explore subnav)
- Settings: categorized tabs with plain descriptions
- WCAG AA contrast on meta text and controls

## Non-goals
- Awwwards marketing motion, hero metrics, card-grid feature marketing
